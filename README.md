Este projeto oferece uma solução simples e eficaz para a automação do processamento de documentos. O objetivo principal é agilizar a conversão de arquivos PDF para formatos mais manipuláveis, garantindo a integridade dos dados durante a transição para HTML e, finalmente, para TXT.

Principais Funcionalidades
Conversão Ágil: Transformação rápida de arquivos PDF complexos.

Pipeline de Dados: Estrutura o fluxo em etapas (PDF > HTML > TXT) para garantir maior precisão na limpeza de dados.

Otimização de Fluxo: Ideal para preparo de datasets ou extração de relatórios automatizados.
